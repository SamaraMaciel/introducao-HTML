Definição de cores

Há várias formas de definição de cores em uma página HTML. Podemos definir uma cor através de cores padrão de navegadores específicos, nesse caso "chamamos" a cor por nome pré-definido (Em todo navegador há cerca de 140 cores pré-definidas). Porém, também podemos definir cores através de:

  • Cores pré-definidas
  • Palavra reservada "currentcolor"(para inserir um novo item com a cor que já foi definida dentro do bloco)
  • RGB
  • RGBA
  • Hexadecimal
  • Hexadecimal com transparência
  • HSL
  • HSLA
  • Inserir grau de transparência com a palavra reservada "transparent"

Padrão RGB:

No padrao RGB(Red, Green e Blue), utilizamos números de 0 a 255 para definir e mesclar tons de vermelho, verde e azul. OBS: 0 corresponde ao preto, e 255 corresponde a cor máxima de cada item RGB. Também é possível definir cada parceça de cor por prtcentagem, embora não seja usual.

Ex: color: rgb(0, 255, 0);
Ex: color: rgb(100%, 50%, 70%);

Padrão RGBA:

É possível atribuir grau de transparência utilizando o padrão RGBA, onde o A indica grau alpha. O grau alpha pode definido entre 0.0 e 1.0, podendo também ser definido no formato: (.0 até 1).

Padrão Hexadecimal:

Por padrão é precedido pelo símbolo # seguido de 6 números alfanuméricos(0 a 9 e a A até F), onde os dois primeiros números representam a parcela de vermelho, os dois números do meio represantam a parcela de verde, e os dois últimos números representam a parcela de azul.

EX: color: #000000; (Representa a cor preta)

EX: color: #FFFFFF; (Representa a cor branca)

OBS: Quando os seis caraceres são iguais também é possível definir a mesma cor de forma "reduzida" representando apenas as três primeiras letras. EX: #FFF; (representa a cor branca).

Para adicionar transparência utilizando o padrão hexadecimal é só acrescentar mais dois dígitos ao final do padrão, com valores entre 00 e FF.

Padrão HSL:

HSL(hue, saturation, lightness) - (matiz, saturaçao, luminosidade).

  • Hue (Matiz): é o grau na roda de cores (de 0 a 360 graus).
    • 0 ou 360 é vermelho
    • 120 é verde
    • 240 é azul
  • Saturation (saturação): valor percentual.
    • 0% é um tom de cinza
    • 100% é a cor total
  • Lightness (luminisidade): luminosidade da cor.
    • 0% é preto
    • 100% é branco

Ex: color: hsl(270, 100%, 50%);

Padrão HSLA:

Assim como no padrçao RGB, no padrão HSL também há a possibilidade de adição do elemento alpha de transparência da cor. Para isso utilizamo a componente alpha (A) que varia de 0 a 1.0, onde 0 significa nenhuma transparência e 1.0 100% de transparência. Profissionais que trabalham com edição de imagens estão mais familiarizados com o padrão HSL.
O mais comum na maioria dos projetos padrão de intenet são o hexadecimal e o RGBA.

Propriedade object-fit: (CSS)

A propriedade "object-fit" do CSS é utilizada para definir o comportamento visual de imagens e vídeos dentro de um elemento/componente (conteiner).

object-fit: fill; padrão

Modelo do código por trás da imagem de exemplo:

<style>
.conteiner{
width: 200px;
height: 300px;
border: 5px solid blue;
}

/*.conteiner, img{
width: 100%;
height: 100%;
}*/

img{
width: 100%;
height: 100%;
}

#container {
object-fit: contain;
}
</style>

<div class="conteiner">
<img src="./Inovação.png">
</div>

OBS: a propriedade object-fit do exemplo acima está recebendo o valor

  • contain (que é um atributo que define que a imagem manterá suas dimensoes, mas de forma que não distorça a imagem) mas também é possível utilizar outros valores de atributo como:
  • fill (que irá parametrizar a imagem para um tamanho prédeterminado pelo dev, e tentará preencher todo o espaço do conteiner, o que provavelmente irá distorcer a imagem.)
  • cover (que redimensiona a imagem para preencher todo o espaço do conteiner, porém ele "corta" as partes da imagem que ficariam para fora do conteiner, dessa a imagem não fica distorcida.)
  • none (esse atributo faz com que a propriedade ignore as dimensões do elemento pai e utile a imagem no seu tamanho original, cortando as "bordas" que ficariam para fora do conteiner, se o tamanho do conteiner for menor do que o tamanho original da imagem.)
  • scale-down (nessa configuração, o esse atributo verifica qual é a menor imagem trazida comparando as características da imagem que seriam retornadas utilizando o contain ou o none, das duas, a configuração que ficar menor será a configuração que o scale-down irá retornar.)

Propriedade object-position:

A propriedade object-position deve ser utilizada junto com a propriedade object-fit, ela é utilizada para definir a posiçao da imagem dentro do conteiner. Com a object-position podemos definir se a imagam firá mais a esquerda, a direita, recuada, etc dentro do conteiner. Valores:

  • porcentagem;
  • pixel;
  • palavras reservadas + valor: right, center, left, bottom, right up, right bottom, start, end, etc.
OBS: É possivel misturar porcentagem, pixel e palavras reservadas + valor na definição da propriedade.

Alterando o fundo dos elementos e trabalhando com estilizaçao de background:

Existem várias formas para se trabalhar com fundos de imagem/elementos em uma página HTML utilizando CSS, a propriedade backgound-color é apeas uma delas. Para mais detalhes e aprofundamento pesquisar na documentação.

  • backgorund-image: url('../diretório'); *ou url do servidor onde a imagem se encontra.
  • back-ground-color: palevioletred;
  • background-image: radial-gradient(lightpink, paleviolet, purple); *Cria um efeito gradiente radial com as cores definidas.
  • background-image: repeating-gradient(lightpink, paleviolet, purple); *Cria um efeito de lista de cores que repete (listrado).
  • backgorund-image: url('../diretório'), url('../diretorio'); *é possível combinar imagens dessa forma, lembrando ue a sequencia inserida das imagsn define também a camada que a imagem vai ficar, a primeira imagem vai ficar na camada superior, a segunda na camada inferior a primeira e assim sucetivamente.
  • OBS: pode ser trabalhado com várias extençoes de imagem, jpeg, png, gif, etc.

Site com modelos de padrões de imagem e seus códigos para reprodução: https://projects.verou.me/css3patterns/

É possível alterar o tamanho de imagens de fundo com a propriedade background-size:

  • background-size: auto; *Define um tamaho de forma automática para a imagem.
  • background-size: cover; *Reajusta a imgagem para cobrir todo o conteiner, mas cuidado se a imagem for muito garnde pode cortar um pedaço e se for muito peuena pode expandir muito perdendo a qualidade.
  • background-size: contain; *Redimensiona a imagem para caber inteira dentro do elemento, caso sobre espaço no elemento a imagem irá começar a se repetir por padrão. Vamos ver posteriormente uma propriedade para evitar esse comportamento de repetição.
  • background-size: 50% 200px; *É possivel utilizar valores para definr o dimensionamento da imagem com essa propriedade. Se for passado apenas um valor, será intendido como parâmetro de largura somente.
  • background-size: 200px 50%, cover; *Nesse exemplo estamos trabalhando com camadas de imagens(duas ou mais imagens em camada). Nesse caso, passamos mais de um caminho de imagem com a propriedade backgorund-image e definimos as configurações de cada imagem separamente dentro do mesmo comando background-size separando por vírgula.

Configurando a repetição de imagem com o a propriedade background-repeat:

  • background-repeat: repeat; *Funciona como padrão, repete a imagem de forma a preencher todo o container.
  • background-repeat: repeat-x; *Repete a imagem apenas na horizontal. Caso "sobre" espaço na parte de baixo do conteiner, vai ficar em branco.
  • background-repeat: repeat-y; *Tem a maesma função do item anterior, porém aqui a imagem só se repete no eixo y (vertical).
  • background-repeat: space; *Repete a imagem tanto horizontalmente quanto verticalmente se cortar a imagem, essa propriedade cria um espaço em branco tanto na vertical quanto na horizontal entre as imagens no conteiner sem cortes na imagem.
  • background-repeat: round; *Redimensiona a imagem para ue ela repita de forma a ocupar todo o espaço do conteiner, sem espaço entre as imagens e sem cortar nenhum pedaço da imagem.
  • background-repeat: no-repeat; *Não repete a imagem, mesmo se ele ficar bem pequena dentro do conteiner.
  • background-repeat: round sapce; *É possível combinar os valores, separando-os por espaço, para definir a repetição nos eixos separadamente.

Ajustando a posição da imagem de fundo com background-position

  • background-position: center 80%; *Basicamente se define posição nos eixos x e y através de valor ou palavras reservadas de posição. Para configuração de múltiplas imagens é só separar a camada por vírgula. Outro exemplo de configuração: background-position: bottom 20px right 10px;.
    OBS: Também é possível trabalhar a posição com gradientes.

Propriedade background-attachment

Essa propriedade define o "movimento" da imagem dentro da área de trabalho, útil principalmente ao utilizar barras de rolagem.
Pesquisar mais sobre efeitos de paralax e outros efeitos com imagem.

  • background-attachment: fixed; *A imagem fica com a posiçao fixa na página, mesmo utilizando a rolagem da página. Aqui prduz um efeito interessante utilizando uma barra de rolagem dentro de um conteiner e outra barra de rolagem para a página em si, dá o efeito de a imagem estar "subindo com a barra de rolagem da página e com a barra de rolagem do conteiner se visualiza a rolagem apenas do texto.
  • background-attachment: scroll; *Com relação ao texto e a imagem dentro de um cointeiner, aplica o mesmo efeito, porém, com a barra de rolagem da página a imagem fica fixa, ou seja, rola junto com a barra de rolagem.
  • background-attachment: local; *A imagem rola junto com o conteúdo em texto inserido no conteiner ao aplicar uma barra de rolagem para esse conteiner. Ao utilizar a barra de rolagem da página a imagem fica fixa.

Propriedade background-origin

Com essa propriedade podemos definir o enquadramento da imagem com relação ao padding interno ao conteiner e a borda.

  • background-origin: padding-box; *A imagem fica enquadrada dentro do conteiner mas sem ultrapassar a linha interna da borda.
  • background-origin: border-box; *A imagem fica enquadara dentro do conteiner entrando na totalidade da faixa da borda.
  • background-origin: content-box; *A imagem fica enquadara no centro do conteiner, sem avaçar pelo padding do conteiner ou pela faixa da borda definida.

Propriedade background-clip

Com essa propriedade podemos definir a área de cobertura não de imagens dentro de um conteiner, mas também a área de cobertura de uma cor da fundo aplicada.

  • background-clip: padding-box; *A imagem ou a cor de fundo cobrem todo i espaçamento interno do conteiner icluindo o padding, se houver, mas não conre a área da borda se esta for definida.
  • background-clip: border-box; *A imagem ou a cor de fundo preencher todo o espaçamento interno do conteiner, inclusive a área da borda, se houver.
  • background-clip: content-box; *Preenche apenas a área interna do conteiner, área após a borda e o padding.
  • background-clip: text; *Aplica o efeito da cor ou da imagem dentro das letras do texto.
    OBS: para esse efito funcionar no Chrome, é necessário escrever o código como: -webkit-background-clip: text;

Propriedade background

É possível definir quase todas as propriedades de fundo anteriores só utilizando o background-color. Inserindo os atributos das propriedades anteriores separados por espaço. Para criar camadas é só adicionar os atributos entre vírgula. Bordas:

A propriedade border assim como a propriedade background pode ser utilizada para definir várias configurações de outras propriedades de borda, é como se fosse um atalho. Mas há também as propriedades específicas de borda:

  • border-width: 5px; *Define a espessura da borda. Outra maneira de definir a espeçura seria: border-width: 5px 10px 25px 30px; Representando respectivamente topo, direita, fundo, esquerda. Também é possívem representar por border-width: 10px 20px 30px; Representado respectivamente topo, laterais e fundo.
    OBS: Para que a página entenda e aplique a espessura, primeiro é preciso definir a propriedade border-style.
    Além de pixel também possível definir a espessura através de palavras chave, como:
    • thin; *Equivale a 1px
    • medium; *Equivale a 3px
    • thick; *Equivale a 5px
  • border-style: ; *Define o estilo da borda (também é possível combinar estilos). Algumas palavra chave que pode receber são:
    • border-style: solid;
    • border-style: dashed; *Pontilhado com traços
    • border-style: dotted; *Pontilhado com pontos redondos
    • border-style: double; *Margem com linha dupla
    • border-style: groove; *Traz uma linha com efeito 3D
    • border-style: ridge; *Traz uma linha com efeito 3D, é parecido com o groove mas inverte as cores
    • border-style: inset; *Dá a impressão que o elemento está "afundado" na tela
    • border-style: outset; *Dá a impressão que o elemento está "saindo da tela
  • borde-color: ; *Define a cor da borda(é possível combinar cores). OBS: Se esse elemento não for definido, a borda ficará com a mesma cor do texto. A cor pode ser definida através do nome da cor, RGB, etc.
  • É possível combinar as configurações utilizando apenas o border. Ex: border-bottom-style: #red dashed;
  • border-radius: 20px 10px; *Cria diferentes agulos de cantos arredondados de bordas, pode receber 4 valores(um para cada borda) representado o sentido horário começando no canto superior direito. Essa propriedade também recebe valores em porcentagem, então para fazer um elemento quardado ficar redondo, é só definir o valor de 50%.
  • border-image *É possível utilizar imagens para definir bordas. Mas a propriedade border-image, assim como a propriedade border é uma shorthand(abreviação). Segue algumas de suas variações:
    • border-image-source: url('../images/exemplo.png'); *Para a definição do caminho da imagem funcionar como borda, é imprescindível que se defina a propriedade border antes. (Ex: border: 30px solid; border-image-source: ...;
      OBS: Também é possível definir a fonte da imagem como gradiente. Ex: border-image-source: linear-gradient(red, blue);. Essa propriedade também pode receber o valor none.
    • border-image-slice: 169; *Essa propiedade "fatia" uma imagem em 9 seções retangulares para poder distribuí-las nas bordas, onde as seções 1, 2, 3 e 4 são "quadrados" fatiados respectivamente do topo direito, topo esquerdo, fundo esquerdo e fundo direito da imagem. As fatias 5, 6 7 e 8 são os "retângulos intermediários que sobraram respectvimente entre as fatias de 1 a 4. E a fatia 9 é o retângulo/quadrado que sobrou no meio da imagem. Para que essa propriedade retorne o efeito esperado, é necessário que o dev teste o valor definido utilizando dev tools do chrome. Essa propriedade também é frequente utilizada em conjunto com a propriedade border-image-repeat, que deve ser definida antes. (Ex: border-image-repeat: repeat; border-image-slice: 169 fill;
      No exemplo acima temos o valor definido fill, ao definir esse valor a seção 9 do slice, que por padrão é descartada, é inserida no conteiner como imagem de fundo. Se já houver uma imagem de fundo definida, ele vai sobrepor.
    • border-image-width: ; *Define a largura da imagem no elemento, e pode recer de 1 a 4 valores, um para todas as bordas ou um valor para cada borda. Aqui também é aceito a palavra chave auto. Valores relativos também são interessantes aqui.
    • border-image-repeat: ; *Basicamente divide a área da imagem em 9 slices para caber na borda. Pode receber 4 valores: repeat (trabalha com a repetição das seções 5, 6, 7 e 8), strech (estica um slice da imagem para caber, também trabalha com os sices 5, 6, 7 e 8), round (redimensiona as seções 5, 6, 7 e 8 pra que a imagem caiba, repetida ou não, por completo), space(cria um espaço entre os trechos de imagem para que eles caibam por completo, também trabalha com as seções 5, 6, 7 e 8).
      OBS: essas palavras chave também podem ser combinadas para criar o efeito desejado.
    • border-image-outset: 10px 20px 30px 40px; *DEfine a distância da borda que a imagem vai ficar da borda. Só recebe valores positivos.
    • border-image: url('...) 169 / 10px / 1 round; *Utilizando essa propriedade para definir todas as outras. No exemplo, respectivamente estão edfinidos os valores para: source, slice, width, outset e repeat. OBS: a / está separando o atributo.
