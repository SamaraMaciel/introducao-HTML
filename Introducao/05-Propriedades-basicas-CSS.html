Propriedades básicas do CSS

Atributos de estilização e exemplos de valores de declaração:

  • Largura: width

    Valores:

    • auto -> valor padrão
    • definixão em píxel (px) -> Ex: 200px
    • inherit -> "copia" a largura definida para o bloco de estilo anterior.
    • definição em porcentagem -> 50%
    • definição em centímetros -> 100cm
    • etc....

  • Altura: height

    Basicamente podemos utlizar os mesmos tipos de declaração de valores que usamos no width.

  • Largura e altura máxima

    Podem ser utilizados para responsividade do página.

    • min-width
    • max-width
    • min-height
    • max-height

  • Margem: margin

    Cria um espaçamento em volta dos elementos. O valor é aplicado dos 4 lados do elemento.

    • Um valor: aplica esse valor dos quatro lados do elemento.
    • Dois valores (separados por espaço): aplica os valores em cima e em baixo do elemento respectivamente.
    • Três valores (separados por espaço): aplica em cima, lados e em baixo do elemento respectivamente.
    • Quatro valores (separados por espaço): aplica em cima, direita, em baixo, esquerda do elemento respectivamente.
    • OBS1: Segue o sentido horário.
    • OBS2: margin: auto; centraliza o elemento apenas na horizontal.
    • OBS3: É possível utilizar valores negativos para margem.

    Variações:

    • margin-top
    • margin-botton
    • margin-left
    • margin-right

  • Padding: É um espaçamento interno do elemento, enquanto que a margem é um espaçamento externo do elemento. É bem útil para criarmos cards, por exemplo.
    Utiliza os mesmos tipos de propriedade que o margin.

  • Box-sizing: Ao aumentar o padding, o tamanho do elemento pode aumentar junto, isso porque o atributo width é calculado como: largura + borda + padding. Para fazer com que o elemento respeite o limite estipulado para width e height, definimos o atributo "box-sizing".
    Esse atributo pode receber dois valores:

    • context-box: se compora de forma padrão, soma os valores de largura + borda + padding.
    • border-box: Redimenciona a largura e a altura do elemento para respeitar o width e height pre definidos.
