Estrutura de declaração de CSS

Os comandos em CCS possuem estrutura Propriedade: valor;.

Formas de declaração de CSS

Há 3 formas de declarar CSS:

  1. In Line:

    2. Através da tag <style> e inserindo os atributos de CSS, como por exemplo: background: aquamarie;
    Essa forma de declaração não é muito utilizada, porque não permite a reutilização de código, mas essa tag possui prioridade sobre outros estilos. Também é utilizado para testes.

  2. CSS interno:

    3. Aqui o código CCS é adicionado dentro da tag <head> da página HTML. Dentro da tag <head> adicionamos outra tag, a <style>, e colocamos as regras CSS nessa área.
    OBS: Nessa modalidade de inserção de CSS precisamos utilizar os "seletores" para definir quais itens receberão essa formatação em CSS.

  3. CSS externo:

    4. Nessa modalidade criamos um arquivo separado com extensão ".CSS" com todas as regras CSS que desejamos aplicar. Esse arquivo é referenciado em todas as páginas HTML, que queremos o mesmo padrão de estilos, através da tag <link>.
    OBS: Vamos chamar essa tag <link> dentro da tag <head> da nossa página HTML.
    Por convensão, dentro da pasta do projeto da página HTML criamos uma pasta chamada "Assets". Dentro dessa pasta criamos subpastas para inserir CSS externo e arquivos javascript.
    Exemplo de declaração da tag <link> com o caminho do arquivo .CSS:
    <link> rel="stylesheet" href="./assets/css/style.css"
    Onde: "rel" é a relação entre o que está sendo referenciado e "href" é o caminho do arquivo externo.
    Pontos positivos de utilização do CSS externo:

    • O carregamento da página fica mais rápido;
    • É possível reaproveitar o estilo do CSS para várias páginas sem precisar reescrever código.
    Pontos negativos de utilização do CSS externo:
    • A página pode ficar temparariamente sem estilo, dependendo da qualidade da conecção com a internet, isso devido ao carregamento do CSS ser feito de forma indepemdente da estrutrutura HTML da página.

Depurando o CSS

Podemos utilizar o "dev tools" do próprio navegador para depurar.

Seletores CSS

Para especificar-mos quais elementos da página HTML queremos estilizar com um bloco específico de CSS, utilizamos os seletores.
Abaixo seguem os tipos de seletores que temos em CSS:

  1. Seletor por tipo:

    2. Busca elementos por uma tag HTML específica;
    Exemplo: É possível aplicar um CSS apenas para uma tag específica ou para um conjunto de tags "filhas" inseridas dentro de uma tag "mãe" (atavés da tag <div> por exemplo).

    • h1 {
      background: blue;
      }
    • div {
      background: blue;
      }

  2. Seletor por ID (#):

    3. Busca elementos atrés do atributo "id". Um "id" é atributo de uma tag, ele idetifica essa tag de forma única no arquivo HTML. Pode receber como valor texto ou número.
    OBS:Um ID deve ser único por página!
    Exemplo:

    • Código HTML: <p id="texto-boas-vindas">Bem vindo!</p>
    • CSS: #texto-boas-vindas {
      background: red;
      }

  3. Seletor por classe (.):

    4. Busca elementos através do atributo "class". Uma "class" também é um atributo de uma tag, esse atributo é utilizado para definir um conjunto de elementos e, diferente do "id", a mesma classe pode e deve ser atribuída mais de uma vez dentro de um arquivo HTML, de acordo com a necessidade.
    Exemplo:

    • Código HTML: <li class="seletores-mais-comuns texto-maisculo">Seletor por classe.</li>
      OBS:Foram inseridas duas classes nesse elemento.
    • CSS:
      .seletores-mais-comuns{
      color: red;
      }
      .texto-maiusculo {
      text-transform: uppercase;
      }

  4. Seletor Universal (*):

    5. Seleciona todos os elementos HTML, independente de qualquer atributo desses elementos. É util para definir características padrão, como por exemplo o tipo de fonte da página, e para aplicar reset CSS.
    Exemplo:

    • CSS: * {
      font-weight: bold;
      }

  5. Seletor de atributo ([atrib])

    6. Seleciona elementos que possuem um atributo específico no arquivo HTML. É possível também procurar atributos com valor específico
    Exemplo:

    • Código HTML: <li> <a title="Google" href="https://www.google.com">Google</a></li>
    • CSS:
      [title] {
      color: red;
      }
    • CSS para estilizar atributo com valor específico:
      [title="Google"] {
      color: red;
      }
      OBS: Com essa estrutura o valor do atributo precisa ser exato!
    • CSS para estilizar atributo buscando valor aproximado:
      • HTML: <li> <a title="site do Google" href="https://www.google.com">Google</a></li>
      • CSS:
        [title~="Google"] {
        color: red;
        }
        OBS: Com essa estrutura "~=", eu busco o valor aproximado não importando a ordem que ele apareça no nome.
    • CSS para estilizar atributo com valor imediatamente seguido de "hífem":
      • HTML: <li> <a title="site-do-Google" href="https://www.google.com">Google</a></li>
      • CSS:
        [title|="site"] {
        color: red;
        }
        OBS: Com essa estrutura "|=" eu busco o atributo que tem o valor que começa com "site" imediatamente seguido de "-" (hífem). Caso o texto procurado não esteja no início do nome do atributo, ou não possua hífem logo após, a busca não irá funcionar.
    • CSS para estilizar atributo buscando o valor por prefixo:
      • HTML: <li> <a title="site-seguro" href="https://www.google.com">Google</a></li>
        <li> <a title="site-nao-seguro" href="http://www.google.com">Google</a></li>
      • CSS:
        [href^="https://"] {
        color: red;
        }
        OBS: Procura todo atributo "href" cujo valor começa especificamente com "https://".
    • CSS para estilizar atributo buscando o valor por sufixo:
      • HTML: <li> <a title="site-seguro" href="https://www.google.com">Google</a></li>
        <li> <a title="site-nao-seguro" href="http://www.google.com">Google</a></li>
      • CSS:
        [href$=".com"] {
        color: red;
        }
        OBS: Procura todo atributo "href" cujo valor termina especificamente com ".com".
    • CSS para estilizar atributo buscando o valor do atributo com uma palavra(caracter) em qualquer posição dentro do valor:
      • HTML: <li> <a title="site-seguro" href="https://www.google.com">Google</a></li>
        <li> <a title="site-nao-seguro" href="http://www.google.com">Google</a></li>
      • CSS:
        [title*="seg"] {
        color: red;
        }
        OBS: Procura todo atributo "href" que contenha "seg" dentro do valor.

Agrupando seletores

É possível agrupar diferentes tipos de seletores para apliar as mesmas regras CSS, para esse agrupamentos separamos os seletores por vírgula (,).
Ex1: .texto, h1, #link-referencia {
color: red;
}
OBS: Essa vírgula funciona como um texto lógico (ou/or).

EX2: .texto.teste
EX3: p.teste
OBS: Nos exmplos 2 e 3 estamos aplicando o teste lógico (e/and). Aqui utilizamos a combinação de seletores sem espaços.

Combinadores

Combinadores definem qual é o tipo de selação entre dois seletores.

  • Combinador descendente (espaço)
    • Ex: li li { color: blue; }
    • Estrutira de código para esse combinador descendente:
      <li> "texto do item da lista"
      <ul>
      <li> "Texto do subitem da lista"
      </li>
      </ul>
      </li>
  • Combinador filho (>)
    • Indica que o segundo seletor da combinação é filho imediato do seletor da esquerda.
    • Ex: div > p
      OBS: Se houver uma tag "p" dentro de uma segunda tag inserida em div (span, por exemplo), essa tag "p" está dentro de div, mas é neta de div e não filha.
  • Combinador Adjacente (+)
    • Indica o seletor irmão imediato do seletor à esquerda.
    • EX: Em uma estrutura -> <div> ... </div>
      <p> ... </p>
      a tag "p" indicada acima é irmã da tag "div".
    • Estrutura: div + p {...}
  • Combinador irmão em geral (~)
    • Indica todos os seletores irmãos do seletor à esquerda (tags de mesma identação).
    • Estrutura: div ~ p {...}

OBS: É possível fazer a combinador de todos os seletores e combinadores apresenados.
